This project is part of the **Database Management Systems 2** course.

It demonstrates the use of:
- **ODBC (pyodbc)** for direct SQL access
- **SQLAlchemy ORM** for object-relational mapping
- **MS SQL Server** as the backend database

## Implemented Tasks

### ORM (SQLAlchemy)
- Display of orders using ORM
- Creation and planning of orders
- Completion of orders with duration and travel time
- Display of spare parts list
- Display of installed spare parts per order
- Completion of orders including spare part usage
- Stock validation with transaction rollback

## Notes
- Database credentials are **not included** in this repository.
- A valid MS SQL Server database with the provided schema is required.
- The project is designed as a **console application**.

## Technologies
- Python 3
- SQLAlchemy
- pyodbc
- MS SQL Server
